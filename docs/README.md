# Lunch Table Games Docs

The docs tree is split by ownership so platform, product, and program work do
not blur together.

`docs/platform/` owns the reusable Lunch Table Games library architecture,
agent parity, generated-game authoring, GitHub setup, external integrations,
and release process.

- [Architecture](./platform/ARCHITECTURE.md)
- [Agent parity](./platform/AGENT_PARITY.md)
- [Generated game authoring](./platform/GENERATED_GAME_AUTHORING.md)
- [GitHub setup](./platform/GITHUB_SETUP.md)
- [Milady and external agents](./platform/MILADY_INTEGRATION.md)
- [Release](./platform/RELEASE.md)
