# @lunchtable/games-side-scroller

Deterministic Lunch Table Games side-scroller engine primitives for generated
runner games, agent parity, physics ticks, collision/objective events, tabletop
components, and renderer-neutral scenes.

```ts
import {
  createSideScrollerRuleset,
  sideScrollerStarterConfig,
} from "@lunchtable/games-side-scroller";
```

This package is browser-first and renderer-neutral. It depends only on the
generic Lunch Table Games core, tabletop, and render packages.
